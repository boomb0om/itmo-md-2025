# Первое ДЗ — поднять инфраструктуру и написать приложение сбора информации

**Работающее в онлайне приложение**, со сбором данных по выбранному кейсу.  
Например, кейс о погоде – ходим в API Яндекс Погоды и собираем информацию в интересном городе, складываем данные в MongoDB каждые N минут.

**Работающая инфраструктура** – работающие PostgreSQL и MongoDB, к которым можно подключиться и сделать запросы, AirFlow с доступом к WebUI – все с использованием CICD в Gitlab / Github

**Первый EL процесс** – с перекладкой данных из MongoDB в PostgreSQL.  
Этот EL процесс нужно сделать без DBT, просто c Python / чем-то еще на ваш вкус.
